Hello,
This repository contains various scripts that I created for an analysis of Franklin St, located in the Chicago Loop
The purpose of each script is as follows:
- 311Data.R: Analysis of 311 requests made on Franklin St
- CrashPeerComparison.R: Compares crash statistics on each of the main streets in the Loop
- DivvyByStation.R: Ridership analysis of Divvy trips on Franklin, separated by station
- DivvyFranklinRidership.R: More generalized analysis of Divvy ridership on Franklin
- ExploratoryAnalyses.R: Initial exploartions of different datasets conducted at the beginning of the project
- FranklinCrashes.R: Crash analysis specifically on Franklin
- LEHD.R: Multimodel travel to the Loop
